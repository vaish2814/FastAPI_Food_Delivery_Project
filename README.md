# 🍔 Food Delivery Backend - FastAPI
**Internship:** Feb 2026 | **Submission Date:** March 22, 2026

## 🚀 Project Overview
This is a comprehensive FastAPI backend developed for a Food Delivery application. It covers 20 specific tasks including CRUD operations, Pydantic validation, and advanced search/sort/pagination logic.

## 🛠 Features
- **Validation:** Strict data types using Pydantic.
- **Workflow:** Real-world order placement logic with total price calculation.
- **Advanced Browsing:** Combined Search, Sort, and Pagination in one endpoint.
